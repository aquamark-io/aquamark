Aquamark API Documentation

This API allows Enterprise users to automate the watermarking process without using the portal UI.

IMPORTANT: Send one PDF at a time. Each file is processed individually to ensure accurate page usage tracking, watermark integrity, and fast response times.

For requests containing multiple files, simply loop through and send each in a separate request. Batch uploads are not supported.

üîê Authentication

Include the API key in the header of every request:

Authorization: Bearer YOUR_API_KEY_HERE

üì• Endpoint: Watermark PDFs

Accepts multipart/form-data with a PDF file and email address. Returns the processed, watermarked PDF as a binary response.

FieldTypeRequiredDescription
user_emailstringYesYour Aquamark account email
filefileYesOne PDF at a time (max 25MB)

Sample Request (cURL)

curl -X POST "https://aquamark-decrypt.onrender.com/watermark" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "user_email=john@example.com" \
  -F "file=@statement1.pdf" \
  --output "statement1 - protected.pdf"

üêç Sample Request (Python)

import requests

url = "https://aquamark-decrypt.onrender.com/watermark"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
files = {
    "file": open("statement.pdf", "rb"),
    "user_email": (None, "john@example.com")
}

response = requests.post(url, headers=headers, files=files)
with open("statement - protected.pdf", "wb") as f:
    f.write(response.content)

üü¶ Sample Request (Node.js)

const axios = require("axios");
const FormData = require("form-data");
const fs = require("fs");

const form = new FormData();
form.append("file", fs.createReadStream("statement.pdf"));
form.append("user_email", "john@example.com");

axios.post("https://aquamark-decrypt.onrender.com/watermark", form, {
  headers: {
    ...form.getHeaders(),
    Authorization: "Bearer YOUR_API_KEY"
  },
  responseType: "stream"
}).then(res => {
  res.data.pipe(fs.createWriteStream("statement - protected.pdf"));
});

üêò Sample Request (PHP - cURL)

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://aquamark-decrypt.onrender.com/watermark");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "Authorization: Bearer YOUR_API_KEY"
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, [
    "user_email" => "john@example.com",
    "file" => new CURLFile("statement.pdf")
]);
$response = curl_exec($ch);
file_put_contents("statement - protected.pdf", $response);
curl_close($ch);
