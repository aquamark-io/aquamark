Aquamark API Documentation

This API allows Enterprise users to upload, protect, and download PDF files - all without using the portal UI.

üîê Authentication

Include the API key in the header of every request:

x-api-key: YOUR_API_KEY_HERE

üì• Endpoint: Watermark PDFs

POST https://aquamark-api.onrender.com/watermark

Returns watermarked PDFs

FieldTypeRequiredDescription
emailstringYesYour Aquamark account email
filesfileYesOne or more PDF files (max 25MB total)

Sample Request (cURL)

curl -X POST https://aquamark-api.onrender.com/watermark \
  -H "x-api-key: YOUR_API_KEY" \
  -F "email=john@example.com" \
  -F "files=@statement1.pdf" \
  -F "files=@statement2.pdf"

  "status": "success",
  "protected_files": [

      "original": "statement1.pdf",
