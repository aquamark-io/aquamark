Aquamark + Salesforce

No managed package — watermark PDFs instantly using our API + APEX

Requirements

  • API access must be enabled for your Salesforce org
  • Add this URL in Remote Site Settings: https://aquamark-decrypt.onrender.com
  • PDF must be converted to Base64 using EncodingUtil.base64Encode(file.Body)

Step 1: Prepare Your PDF

Use EncodingUtil.base64Encode(file.Body) to convert your PDF to a base64 string.

Step 2: Call the Aquamark API

Paste this method into an Apex class. No extra setup required.

// Sample Apex Callout
HttpRequest req = new HttpRequest();
req.setEndpoint('https://aquamark-decrypt.onrender.com/watermark');
req.setMethod('POST');
req.setHeader('Content-Type', 'application/json');
req.setHeader('Authorization', 'Bearer YOUR_API_KEY'); // Replace with your key

Map body = new Map{
  'user_email' => 'you@example.com',
  'file' => 'BASE64_ENCODED_PDF',
  'lender' => 'Fora Financial' // optional
};

req.setBody(JSON.serialize(body));

Http http = new Http();
HttpResponse res = http.send(req);
System.debug(res.getBody());

Step 3: Use Anywhere

Call from a Flow using Apex Actions, from a trigger, or any custom logic.

Optional: Save the Protected File in Salesforce

To attach the watermarked file to a record (e.g. Opportunity, Custom Object), use:

// Save returned file as an Attachment
Attachment a = new Attachment();
a.Name = 'Watermarked - protected.pdf';
a.Body = EncodingUtil.base64Decode(res.getBody());
a.ParentId = 'RECORD_ID_HERE';
insert a;

Optional: Auto-Delete File After Submission

To reduce storage usage, you can auto-delete the file after a short delay. Set up a Scheduled Apex job like this:

// Example: Delete after 1 hour (custom class required)
global class DeleteOldFiles implements Schedulable {
  global void execute(SchedulableContext sc) {
    List filesToDelete = [SELECT Id FROM Attachment WHERE CreatedDate <= :System.now().addMinutes(-60)];
    delete filesToDelete;
  }
}
// Schedule via Setup > Apex Classes > Schedule Apex
