Aquamark + Salesforce

No managed package needed ‚Äî instantly watermark PDFs using our API + APEX.

Requirements

  • API access must be enabled for your Salesforce org
  • Add this URL in Remote Site Settings: https://salesforce-proxy-kth9.onrender.com
  • PDF must be converted to Base64 using EncodingUtil.base64Encode(fileBody)

Step 1: Prepare Your PDF

Use EncodingUtil.base64Encode(fileBody) to convert your PDF to a base64 string.

Step 2: Call the Aquamark API

Paste this method into an Apex class. No extra setup required.

// Sample Apex Callout
HttpRequest req = new HttpRequest();
req.setEndpoint('https://salesforce-proxy-kth9.onrender.com/watermark');
req.setMethod('POST');
req.setHeader('Content-Type', 'application/json');

Map<String, Object> body = new Map<String, Object>{
  'user_email' => 'you@example.com',
  'file' => 'BASE64_ENCODED_PDF',
  'lender' => 'Optional Lender Name'
};

req.setBody(JSON.serialize(body));
Http http = new Http();
HttpResponse res = http.send(req);

System.debug('Response status: ' + res.getStatus());
System.debug('Response body: ' + res.getBody());
