Simple API Integration

Aquamark's integration is lightweight and straightforward.
Simply intercept the PDF download event, send files to our API endpoint, and return the watermarked version to the user.

Endpoint

POST https://aquamark-lenders.onrender.com/watermark

Headers

Authorization: Bearer YOUR_API_KEY

Request Format

Use multipart/form-data with the following fields:

Field Type Required Notes
file File Required The original PDF to protect
name String Required Person signed in and initiating download
company String Required Your company name

Response

  • Returns a 200 OK with the protected PDF as binary data.
  • Client should treat it as a file download.

Common Implementations

  • Salesforce - override the download button or embed a Visualforce page to return the protected file.
  • Custom or proprietary CRMs - intercept the click/download event and POST the file via the API.
