API Documentation

Start watermarking seamlessly in minutes with our simple API

Authentication

Contact hey@aquamark.io to request your API key.

Authorization: Bearer YOUR_API_KEY

Endpoint

POST https://aquamark-decrypt.onrender.com/watermark

Field Type Required Description
user_email string Required Your Aquamark account email
file file Required One PDF file (max 25MB)
state string Recommended State name used to trigger compliance disclaimers. Add a custom compliance field in your CRM to store them.
lender string Optional Lender name to embed in QR code
salesperson string Optional Salesperson name for QR code
processor string Optional Processor name for QR code

Code Examples

cURL

curl -X POST "https://aquamark-decrypt.onrender.com/watermark" \ -H "Authorization: Bearer YOUR_API_KEY" \ -F "user_email=john@example.com" \ -F "file=@statement1.pdf" \ -F "lender=Fora Financial" \ -F "salesperson=Jack Dawson" \ -F "processor=Rose DeWitt" \ -F "state=CA" \ --output "statement1 - protected.pdf"

Node.js

const axios = require("axios"); const fs = require("fs"); const FormData = require("form-data"); const form = new FormData(); form.append("user_email", "john@example.com"); form.append("state", "CA"); form.append("file", fs.createReadStream("./bankstatement.pdf")); axios.post("https://aquamark-decrypt.onrender.com/watermark", form, { headers: { ...form.getHeaders(), Authorization: "Bearer YOUR_API_KEY" }, responseType: "arraybuffer" }).then(response => { fs.writeFileSync("protected.pdf", response.data); const disclaimer = response.headers["x-state-disclaimer"]; console.log("Compliance Disclaimer:", disclaimer); }).catch(error => { console.error("Error:", error.message); });

Python

import requests with open("bankstatement.pdf", "rb") as f: files = { "file": f } data = { "user_email": "john@example.com", "state": "California" } headers = { "Authorization": "Bearer YOUR_API_KEY" } response = requests.post( "https://aquamark-decrypt.onrender.com/watermark", data=data, files=files, headers=headers ) with open("protected.pdf", "wb") as out_file: out_file.write(response.content) disclaimer = response.headers.get("x-state-disclaimer") print("Compliance Disclaimer:", disclaimer)

Postman Setup

  • Method: POST
  • URL: https://aquamark-decrypt.onrender.com/watermark
  • Authorization: Bearer YOUR_API_KEY
  • Body Type: form-data

Successful Response

Returns a protected PDF file with optional compliance header:

X-State-Disclaimer: CFL license and Disclosure required.

Developer Workflow

  1. Send a POST request with your PDF and metadata
  2. Receive a protected PDF + optional disclaimer header
  3. Save the PDF and push state disclaimers to CRM
  • State compliance disclaimers are returned in response header
  • Works with state names or abbreviations (e.g. CA, California)
  • Push disclaimer to your custom compliance CRM field

Important Notes

  • PDF only. One file per request
  • Retail users must upload logo in portal before using the API
