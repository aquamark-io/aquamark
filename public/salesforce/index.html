Salesforce Integration Setup

This guide walks you through connecting your Salesforce instance to Aquamark in just a few steps.

1. Sign up for the Aquamark Enterprise Plan

Go to Aquamark.io and subscribe to the Enterprise plan.

2. Request the Salesforce Integration Kit

Email us at hey@aquamark.io to get the full Apex and Visualforce files.

3. Add Remote Site Settings in Salesforce

In Salesforce Setup, search for Remote Site Settings and add the following URLs:

  • Aquamark Decrypt Server
  • Aquamark Proxy Server
  • Supabase Storage (for logo access)

4. Create Apex Class

In Salesforce Setup, go to Apex Classes, click New, and paste in the provided Apex code.

Edit the following in the code:

  • Your user_email
  • The lender list (optional customization)

5. Add the Visualforce Page

Create a new Visualforce Page and paste in the provided code. Name it something like Aquamark.

6. Embed the Visualforce Page into Opportunity Layout

In the Opportunity page layout editor, drag in the new Visualforce section.

Adjust the section height to ensure the full interface is visible.
